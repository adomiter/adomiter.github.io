Census Data Reporter

This project was inspired by the Census Reporter website. I hope to build another version of this web application for personal use. This new version will allow provide better mapping functions, the ability to compare multiple areas of interest, and the ability to save reports on areas of interest.

The image to the right is a placeholder for the image of the final product.

Features
map Mapping Functions

Users interested in exploratory data analysis can map variables of interest. Users can customize their maps.



compare Comparisons

Users can select up to three locations (census tract, zip code, county, MSA, etc) and compare relevant information. Information is directly accessed through the U.S. Census Bureau API.

save Save reports

Users can download reports on areas of interest. They can also bookmark reports to come back to later.
