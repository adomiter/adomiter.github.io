AI Reports

This project started during my internship with Origin Investments. The purpose of the project was to leverage machine learning and artificial intelligence to empower investors when underwriting multifamily properties. The report is a combination of relevant public data points and forecasts generated using a proprietary model.

Features
insights Rent Growth Forecasts

Predicted 5 year rent growth is provided and broken up by year. These forecasts are made using a proprietary rent growth model.

emoji_people Demographic Data

Demo data for the tract in which the property is located is given. Percentiles are provided for context.

check_box Reports by Type

Users can select the type of report they wish to view. Options include development, value add, or sale.
